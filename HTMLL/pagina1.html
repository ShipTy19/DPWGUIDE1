Definiciones de HTML5, CSS3 y JavaScript

¿Qué es HTML5?

HTML5 es la última versión del lenguaje de marcado utilizado para estructurar páginas web.

¿Qué es CSS3?

CSS3 es un lenguaje de estilos que permite mejorar la apariencia de las páginas web.

¿Qué es JavaScript?

JavaScript es un lenguaje de programación que permite crear interactividad en las páginas web.
